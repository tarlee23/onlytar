    การจ้างงาน

    GRAPHIC


    แพ็คเกจ Graphic
    1. ออกแบบภาพแบนเนอร์ ภาพละ 1,500 - 4,000 บาท (ถ้าลูกค้าไม่เคยมีเทมเพลต CI มาก่อน อาจมีค่าใช้จ่ายเพิ่มเติม)
    2. Logo Design โลโก้การ์ตูน (2,500) Modern/ลายไทย (5,500)
    3. Presentation นักศึกษา / นิสิต (800) Presentation งานวิจัย สำหรับนักศึกษา / นิสิต(1,500) Presentation บริษัท / องค์กร (2,500)
    4. Infographic & Icon ออกแบบอินโฟกราฟิก โดยผู้ว่าจ้างมีเนื้อหาที่ต้องการเตรียมไว้ ผลงานออกมาในลักษณะรูปเดี่ยว (2,500) Data Visual ออกแบบอินโฟกราฟิก โดยผู้ว่าจ้างให้ข้อมูลดิบ หรือข้อมูลจำนวน ที่มีความซับซ้อน ซึ่งจำเป็นในการใช้ระยะเวลาตีความ และออกแบบเชิงลึก (5,500)
    5. Photoshop die cut product set (only product image) (300) retouch&die cut effect(models & product img) ตัดต่อทุกชนิด (500)รีทัชภาพ photomanipulation&Digital art (1,300)

    ขั้นตอนการทำงาน Graphic
    1. รับรายละเอียดงานจากลูกค้า
    2. ทำ Mood&Tone สำหรับงานลูกค้า
    3. เมื่อลูกค้าสรุป Mood&Tone เรียบร้อยแล้ว ทางทีมงานจะเริ่มออกแบบผลงาน
    4. ส่งงานพร้อมแก้ไข
    5. แก้ไขและส่งfinal
    • ระยะเวลาในการออกแบบ 2-3 วัน/งาน
    • หากงานเร่ง ต้องการภายใน 1 วัน คิดราคาเพิ่ม 650 บาท
    • กราฟิกดีไซน์ทุกประเภท แก้ได้ 3 ครั้ง
    • หากมีการแก้เพิ่ม รอบละ 490 บาท
    • สามารถทักมาสอบถามเรทราคาของงานแต่ละประเภทได้เพิ่มเติม

     **หากคอนเฟิร์มงานแล้วมีการแก้ไขภายหลังทางบริษัทจะคิดราคาเท่ากับการสร้างผลงานขึ้นมาใหม่



    PROGRAMMING


    แพ็คเกจ Programming
    1. งานด่วน แก้บัค แก้ข้อผิดพลาด (1,500)
    2. สร้าง Web Application หน้าเดียว ตามความต้องการ (3,000)
    3. สร้าง Web Application ตามความต้องการ (6,000)

    ขั้นตอนการทำงาน Programming
    1. ตรวจสอบ requirement, scope งานของลูกค้า และประเมินระยะเวลาการทำงาน,ราคา
    2. เริ่มทำงานและอัพเดตงานให้ลูกค้าตามที่ตกลงกัน จนกว่าจะส่งมอบงานให้ลูกค้า
    3. การแก้ไขงาน
      • ไฟล์ Source code ทั้งหมด
      • คำแนะนำการใช้งาน Website, Program
      1. ให้คำแนะนำการใช้ Website
      • ระยะเวลาในทำ ขึ้นอยู่กับงานที่ได้รับ
      • หากงานเร่ง ต้องการภายใน 2 อาทิตย์ คิดราคาเพิ่ม 10,000 บาท
      • สามารถแก้ไขได้ 2 ครั้ง
      • หากมีการแก้เพิ่ม รอบละ 1,000 บาท
      • สามารถทักมาสอบถามเรทราคาของงานได้เพิ่มเติม

      **หากคอนเฟิร์มงานแล้วมีการแก้ไขภายหลังทางบริษัทจะคิดราคาเท่ากับการสร้างผลงานขึ้นมาใหม่



    DRAWING



    แนวงานที่รับทำ Drawing
    1. ภาพของขวัญ/FAN ART
    2. ภาพตามสั่ง
    3. งานภาพปก
    4. การวาดงานตามแบบที่ให้มา(วาดตามที่ลูกค้าส่งมาให้)
    • Semi-real style
      • งานรูปแบบเสมือนจริง
      • ＊สามารถสั่งตัดส่วนต่างๆไปขยับได้โดยแจ้งก่อน ไม่งั้นจะตัดเส้นรวมไปเลย＊
      • หัวถึงอก 600 บาท
      • เต็มตัว 800 บาท
      • Full CG เริ่มต้น 1500บาท (ต้องคำนวนความยากง่ายก่อน)
        • 2D Style
        • งานรูปแบบลายเส้นภาพแบบสไตล์อนิเมะ
        • ＊สามารถสั่งตัดส่วนต่างๆไปขยับได้โดยแจ้งก่อน ไม่งั้นจะตัดเส้นรวมไปเลย＊
        • เรทราคา
        • หัวถึงอก 500 บาท
        • เต็มตัว 800 บาท
        • Full CGเริ่มต้น 1200บาท (ต้องคำนวนความยากง่ายก่อน)

          • ขั้นตอนการทำงาน Drawing
          1. พูดคุยกับลูกค้า ตกลงเลือกแพ็คเกจ
          2. ลูกค้าส่งรูปบรีฟรายละเอียดงานมาให้ (jpeg)
          3. เริ่มวาด/ลงสี แล้วส่งให้ดูเป็นระยะๆหรือส่งตอนใกล้สมบูรณ์แล้ว (ส่งเป็นภาพแคปหน้าจอแล้วให้ลูกค้าเลือกแก้ไขเพิ่มเติมตามจุด)
          4. เมื่อภาพเสร็จสมบูรณ์แล้วจะให้ลูกค้าแจ้งที่อยู่จัดส่งชิ้นงาน
          5. ผลิตชิ้นงานเตรียมส่ง กรณีที่รับเป็นสิ่งของ
          6. เสร็จสมบูรณ์
          • ระยะเวลาในการออกแบบ 2-3 วัน/งาน
          • หากงานเร่ง ต้องการภายใน 1 วัน คิดราคาเพิ่ม 650 บาท
          • งานทุกประเภท แก้ได้ 3 ครั้ง
          • หากมีการแก้เพิ่ม รอบละ 490 บาท
          • สามารถทักมาสอบถามเรทราคาของงานได้เพิ่มเติม

          **หากคอนเฟิร์มงานแล้วมีการแก้ไขภายหลังทางบริษัทจะคิดราคาเท่ากับการสร้างผลงานขึ้นมาใหม่



    MOTION GRAPHIC



    แพ็คเกจ Motion Graphic
    1. Motion Graphic 30 วินาที เน้นรูปภาพจริง, ภาพ Screenshot, Platform หรือ Graphic (3,500)
    2. Motion Graphic 1 นาที เน้นเล่าเรื่องราว อธิบายรายละเอียดผ่าน Motion / Animation (6,000)

    ขั้นตอนการทำงาน Motion Graphic
    1. อธิบายลักษณะงาน โทนสี รูปแบบ Mood&Tone พร้อมส่ง script งานให้
    2. ขึ้น storyboard ตาม script งานที่ลูกค้ามอบหมาย
    3. แก้ไขครั้งที่ 1 - แก้ไข storyboard
    4. Motion ตาม storyboard ที่ผ่านแก้ไขแล้ว และ final งานพร้อมเสียงพากษ์และดนตรีประกอบ
    5. แก้ไขครั้งที่ 2 - แก้ไข Motion
    6. ส่งไฟล์งานตามที่ลูกค้าตกลงกันไว้
      • ระยะเวลาในการออกแบบอย่างต่ำ 5 วัน
      • หากงานเร่ง คิดราคาเพิ่ม 2,000 บาท
      • หากมีการแก้เพิ่ม รอบละ 500 บาท
      • สามารถทักมาสอบถามเรทราคาของงานได้เพิ่มเติม

        • **หากคอนเฟิร์มงานแล้วมีการแก้ไขภายหลังทางบริษัทจะคิดราคาเท่ากับการสร้างผลงานขึ้นมาใหม่



    PRODUCTION



    แพ็คเกจ Production
    1. รับผลิตวิดีโอสินค้าหรือบริการ/ วิดีโองานอีเว้นท์/ วิดีโอโมชั่นกราฟฟิก หรืออื่นๆ โดยมีแพ็คเกจต่างๆ ให้ลูกค้าได้เลือกตามงบประมาณที่ลูกค้าพึงพอใจ
    2. แพ็คเกจ Product Video ราคาเริ่มต้น 65,000 บาท
    3. แพ็คเกจ Product Video แบบมีนักแสดง (แพ็คเกจเริ่มต้น) ราคาเริ่มต้น 150,000 บาท
    4. แพ็คเกจ Product Video ราคา 250,000 - 300,000 บาท
    5. แพ็คเกจ Product Video แบบมีนักแสดงราคา 300,000 - 350,000 บาท
    6. แพ็คเกจงานวิดีโอโฆษณา ประเภท Interview Video ราคา 300,000 - 350,000 บาท
    7. แพ็คเกจงานวิดีโอโฆษณา ประเภท Awareness/ Branding Video ราคา 450,000 - 650,000 บาท
    8. แพ็คเกจ Commercial Video (TVC & ONLINE) ราคา 800,000 บาทขึ้นไป

    แนวทางการทำงาน Production
    ด้วยประสบการณ์ของบริษัทรับเหมามีเดีย และทีมผู้ผลิตพร้อมสร้างสรรค์ผลงานคุณภาพระดับอินเตอร์ให้คุณ เราสามารถเป็นตัวช่วยให้คุณได้ในทุกขั้นตอนตั้งแต่ : คิดไอเดีย, Pre-to-Post Production ไปจนถึงการ Deliver ชิ้นงานตาม Spec/Format และ Platform ที่คุณต้องการ เราเชี่ยวชาญในการผลิตงานตั้งแต่สเกลเล็ก-ใหญ่และมุ่งเน้นการผลิต Cinematic Content ที่มีความตื่นตาตื่นใจ สวยงาม มีสไตล์ และตอบโจทย์ของตลาด พวกเราเป็นที่ไว้วางใจโดย ศิลปิน, บริษัท, โปรดักต์ และ แบรนด์ ชั้นนำระดับประเทศ และหวังว่าคุณจะมาร่วมเป็นส่วนหนึ่งเพื่อสร้างสรรค์ผลงานที่แปลกใหม่ไปกับเรา!
